Privacy Policy

Quick Index Checker Chrome Extension

Last updated: June 26, 2026

Summary

Quick Index Checker is a free Chrome extension that checks whether URLs are indexed by Google Search. We do not collect, store, or transmit any personal data. Everything runs locally in your browser.

Data Collection

This extension does not collect or transmit any of the following:

  • Personal information (name, email, etc.)
  • Browsing history or activity
  • URLs you check (they are processed locally and never sent to our servers)
  • Cookies or tracking identifiers
  • Analytics or crash reports

How It Works

When you check a URL's indexing status, the extension sends a search query (site:your-url) directly to Google's search engine from your browser — exactly as if you typed it into Google.com yourself. Google may process this query according to their own Privacy Policy. The extension does not intercept, log, or store these queries on any server we control.

Data Storage

The extension uses chrome.storage.local solely for optional features such as:

  • Pending check URL (passed from right-click context menu to the popup; cleared immediately)
  • Future V2 features: check history and user preferences (stored locally only)

All storage is local to your browser. We have no access to it.

Permissions

The extension requires the following permissions:

  • activeTab — To read the current page URL when you click the extension icon
  • contextMenus — To add the "Check if indexed" right-click option
  • storage — For local-only feature storage (history, preferences)
  • host_permissions (google.com) — To send indexing queries to Google Search

Third-Party Services

When checking a URL, the extension sends a request to google.com. Google's use of that request data is governed by Google's Privacy Policy.

Contact

If you have questions about this privacy policy, please contact the developer at the GitHub repository.
