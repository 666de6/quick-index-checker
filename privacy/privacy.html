Privacy Policy

Quick Index Checker · Last updated: July 3, 2026

Summary

Quick Index Checker is a free Chrome extension that checks whether URLs are indexed by Google Search. We do not collect, store, or transmit any personal data. Everything runs locally in your browser.

Data Collection

This extension does not collect or transmit:

  • Personal information (name, email, etc.)
  • Browsing history or activity
  • URLs you check (processed locally, never sent to any server we control)
  • Cookies or tracking identifiers
  • Analytics or crash reports

How It Works

When you check a URL's indexing status, the extension sends a search query (site:your-url) directly to Google's search engine from your browser — exactly as if you typed it into Google.com yourself.

Google may process this query according to their own Privacy Policy. The extension does not intercept, log, or store these queries on any server we control.

Local Storage

The extension uses chrome.storage.local solely for:

  • Pending check URL (passed from right-click context menu; cleared on read)
  • Check history and preferences (stored locally only, never transmitted)

Permissions

  • activeTab — Read current page URL when you activate the extension
  • contextMenus — Add right-click "Check if indexed" option
  • storage — Local-only history and preferences
  • scripting — Detect noindex meta tags on the current page
  • sidePanel — Display the extension as a side panel drawer
  • https://*/* — Access any website to send indexing queries to Google Search and import sitemaps from any domain

Third-Party Services

The only external requests are to google.com (to check indexing) and any website the user provides a sitemap URL for (to import URLs for bulk checking). Google's use of request data is governed by Google's Privacy Policy.
