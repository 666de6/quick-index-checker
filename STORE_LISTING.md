## Detailed Description
Quick Index Checker is a free, privacy-first Chrome extension that tells you whether any webpage is indexed by Google — in one click.

### 🚀 Key Features

**One-Click Single Check**
Open the side panel, click "Check Index Status" — instantly see if the current page is indexed.

**Bulk URL Check**
Paste up to dozens of URLs and check them all at once. Live progress bar and waterfall-style results show you indexing status as each URL is checked.

**Sitemap Import**
Paste any sitemap.xml URL and auto-import all listed URLs into the bulk checker. Perfect for auditing your entire site.

**Right-Click Context Menu**
Right-click any link or page → "Check if indexed on Google." Opens directly in the side panel.

**CAPTCHA Auto-Resume** 🛡️
If Google asks for verification, we save your progress. Complete the CAPTCHA once, click "Continue Checking," and resume exactly where you left off — no lost work, no starting over. Many index checkers don't handle this at all.

**CSV Export**
Export all results with URL and index status for reporting or further analysis.

**Check History**
All past checks are saved locally with timestamps. Re-check any URL from your history.

### 🔒 100% Private — No Data Collection
- No signup required
- No analytics, no tracking, no cookies
- All checks go directly from your browser to Google — we never see your URLs
- Check history stored locally on your device only
- Full privacy policy: https://666de6.github.io/quick-index-checker/privacy/privacy.html

### ⚡ How It Works
The extension performs a `site:` search on Google for each URL and reads the result directly from the rendered page — the same way you'd check manually, just automated.

### 📋 Permissions Explained
- **activeTab** — Read current page URL when you click the extension
- **contextMenus** — Add right-click "Check if indexed" option
- **storage** — Save check history locally
- **scripting** — Read Google search results in background tabs
- **sidePanel** — Display the extension UI as a convenient side drawer
- **google.com** — Send indexing queries to Google Search

Zero data leaves your browser. No servers, no middlemen.
